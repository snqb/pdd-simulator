# Bishkek Vibe — Asset Wishlist

## Вайб Бишкека (по фоткам)

Что делает Бишкек Бишкеком на перекрёстке:
- **Хрущёвки 4–5 этажей** — бежевые/жёлтые панельки, балконы с бельём, спутниковые тарелки на крышах
- **Тополя** — высокие, узкие, пирамидальные (не круглые дубы)
- **Жёлтая разметка** — зебры жёлто-белые, вафельная разметка на перекрёстках (жёлтый ромб)
- **Маршрутки** — белые/жёлтые микроавтобусы
- **Провода** — троллейбусные/электрические провода над дорогой
- **Дорожные знаки** — русский стандарт (СТОП, пешеход в синем треугольнике)
- **Асфальт** — серый, потрескавшийся, заплатки
- **Тротуар** — брусчатка, не идеальная
- **Машины** — преимущественно белые/серебристые, праворульные (Honda Fit, Toyota)
- **Курьеры** — Яндекс/Glovo рюкзаки

## Текстуры для атмосферы (бесплатные)

Всё ищется через "ТЕМА seamless texture free":
- **Асфальт**: `cracked asphalt seamless PBR` → [ambientCG.com](https://ambientcg.com/list?q=asphalt) (CC0)
- **Брусчатка**: `cobblestone pavement PBR` → ambientCG
- **Стены зданий**: `concrete panel wall PBR` → ambientCG
- **Трава**: `grass PBR` → ambientCG

[ambientCG.com](https://ambientcg.com/) — CC0 PBR текстуры (albedo + normal + roughness). Идеально для Three.js.

## Pipeline скачивания

Sketchfab нужен **API token** для download (бесплатная регистрация):
```bash
# 1. Зарегистрироваться на sketchfab.com
# 2. Settings → API Token → скопировать
# 3. Сохранить: pass insert api/sketchfab

# 4. Скачать модель:
TOKEN=$(pass api/sketchfab)
MODEL_ID="05229ac1d1f94e6c8cacaad91110c602"
# Get download URL
curl -s "https://api.sketchfab.com/v3/models/$MODEL_ID/download" \
  -H "Authorization: Token $TOKEN" | jq '.gltf.url'
# Download
curl -sL -o model.zip "URL_FROM_ABOVE"
unzip model.zip
```
