# MVP Roadmap

## Локации для MVP (5 типов перекрёстков)

### 1. ✅ Крестообразный перекрёсток (СДЕЛАНО)
Что есть: светофор, 4 направления, зебры, вафельная разметка.
Бишкек-аналог: Чуй/Манаса.

### 2. 🔲 Т-образный перекрёсток
Главная дорога + примыкание. Знак "Уступи дорогу".
Бишкек-аналог: любой двор выезжающий на проспект.
Новые ассеты: знак 2.4, знак "Главная дорога".

### 3. 🔲 Кольцо (кольцевое движение)
Круг с въездами. Кто кому уступает.
Бишкек-аналог: кольцо у Южных ворот.
Новые ассеты: знак "Круговое движение", curved road segments.

### 4. 🔲 Пешеходный переход (без перекрёстка)
Прямая дорога + нерегулируемая зебра. Пешеход выходит на дорогу.
Бишкек-аналог: переход у школы/рынка.
Новые ассеты: знак "Пешеходный переход", школа/магазин.

### 5. 🔲 Автобусная остановка
Автобус останавливается, пешеходы выходят, машины объезжают.
Бишкек-аналог: любая остановка.
Новые ассеты: остановочный павильон (KayKit `bus_stop` есть!), знак остановки.

## Стили зданий для разных зон Бишкека

| Зона | Здания | Атмосфера |
|---|---|---|
| **Центр** (Чуй, Эркиндик) | Советские + стекло бизнес-центры + кафе | Плотный, оживлённый |
| **Микрорайоны** (3-6-7-12 мкр) | Хрущёвки + панельки 9 этажей | Дворы, тополя, маршрутки |
| **Новостройки** (Южные, Кок-Жар) | Жилые комплексы (ЖК) + частные дома | Широкие дороги, меньше деревьев |
| **Ошский рынок / базар** | Одноэтажные, павильоны, толпа | Хаотичный, много пешеходов |

### Нужные ассеты для разнообразия

| Тип | Что искать | Где |
|---|---|---|
| **Стеклянный офис** | "modern glass building low poly" | Sketchfab / Poly Pizza |
| **ЖК (жилой комплекс)** | "modern residential building" | Sketchfab |
| **Магазин/кафе** | "shop storefront low poly" | KayKit, Poly Pizza |
| **Остановка** | KayKit `bus_stop` | Уже есть в скачанном пакете! |
| **Забор дворовой** | "concrete fence post soviet" | Sketchfab |
| **Остановочный павильон** | "bus shelter modern" | Sketchfab (уже нашли) |
