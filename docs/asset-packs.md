## 4. Люди (Пешеходы, Велосипедисты) 🚶

### Quaternius Ultimate Animated Characters ⭐
- **URL:** https://quaternius.com/packs/ultimatedanimatedcharacter.html
- **Download:** https://drive.google.com/drive/folders/1sNi1AfenfPRrvRt5yfaj5QMMd6KKcUJ5
- **Что:** 50+ анимированных персонажей с анимациями (ходьба, бег, idle)
- **Лицензия:** CC0
- **Формат:** FBX, OBJ, Blend → конвертить в GLTF
- **Вердикт:** ⭐ ЛУЧШИЙ бесплатный вариант для анимированных пешеходов.

### Mixamo (Adobe)
- **URL:** https://www.mixamo.com/
- **Что:** Бесплатные анимации для любого персонажа. Upload модель → получи walking/cycling/running анимацию.
- **Лицензия:** Бесплатно для коммерческого использования
- **Формат:** FBX → конвертить в GLTF через Blender
- **Workflow:** Quaternius character → Mixamo → walking animation → GLTF → Three.js
- **Вердикт:** ⭐ Обязательно для анимации ходьбы.

### Ready Player Me
- **URL:** https://readyplayer.me/
- **Что:** Кастомные 3D-аватары, каждый уникальный
- **Формат:** GLB (нативно!)
- **Анимация:** Через Mixamo или встроенные
- **Вердикт:** Если нужны реалистичные уникальные пешеходы.

## Рекомендуемый Pipeline

```
1. ЗДАНИЯ:
   Скачать Soviet Apartment Building со Sketchfab (GLTF)
   → Заменить Kenney suburban домики
   → Добавить 3–4 хрущёвки по углам перекрёстка

2. ЛЮДИ:
   Скачать Quaternius Animated Characters
   → Blender: конвертировать в GLTF с анимациями
   → Или: Mixamo → walking/cycling animation → GLTF
   → Three.js: AnimationMixer для воспроизведения

3. МАШИНЫ:
   Kenney Car Kit — уже есть и работает ✅
   → Подключить colormap.png текстуру (сейчас белые)

4. ДОРОГИ:
   Kenney Roads — уже есть ✅
   → Дополнить Quaternius Modular Streets если нужна модульность

5. ДЕРЕВЬЯ:
   Починить текстуры существующих моделей
   → Или заменить на Quaternius nature pack
```

## Лицензии

| Лицензия | Что можно | Нужно ли указывать автора |
|---|---|---|
| **CC0** | Всё. Коммерция, модификация, без ограничений | Нет |
| **CC-BY** | Всё, но надо указать автора | Да (в credits) |
| **CC-BY-NC** | Только некоммерчески | Да |
| **Free Standard** (Sketchfab) | Зависит от условий | Проверить |

Все рекомендованные паки — CC0 или CC-BY. Для коммерческого ПДД-продукта — ✅.
