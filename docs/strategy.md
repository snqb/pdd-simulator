# ПДД Video Production — Strategy

## Задача

Производить видео-уроки по ПДД: перекрёстки, знаки, приоритеты, пешеходы.
Нужно чтобы люди двигались, машины ехали, курьеры проезжали.
Хронометраж 15–60 сек на сценарий. Визуально — заебато.

## Двухфазная стратегия

### Фаза 1: Kling AI (сейчас → контент за дни)

**Что:** Генерируем AI-видео из готовых иллюстраций ПДД.

**Pipeline:**
```
Иллюстрация ПДД (AI-сгенерированная или нарисованная)
  → Kling 3.0 I2V Pro (Image-to-Video)
  → 5с шоты × N (Multi-Shot для сцены)
  → DaVinci Resolve (склейка, стрелки, озвучка)
  → Готовый урок
```

**Инструменты:**
- [klingai.com](https://klingai.com) — генерация
- Nano Banana / Flux / Midjourney — кейфреймы
- DaVinci Resolve — монтаж

**Бюджет:** ~$50–100 на серию из 20 уроков.

**Ограничения:** Нет точного контроля движения. AI может накосячить с ПДД. Каждый сценарий — ручная работа.

### Фаза 2: Three.js + SUMO (параллельно → платформа)

**Что:** Интерактивный веб-тренажёр с точной симуляцией.

**Pipeline:**
```
SUMO (транспортная симуляция, ПДД-логика)
  → TraCI API (Python ↔ SUMO)
  → WebSocket → Three.js / React Three Fiber
  → 3D-рендер в браузере
  → Интерактив: квизы, выбор, от первого лица
```

**Инструменты:**
- SUMO — симуляция трафика (бесплатно, точнейшая)
- Three.js / R3F — 3D в браузере
- Kenney Assets / Sketchfab — 3D-модели
- Next.js — платформа

**Бюджет:** Dev time (2–4 недели для MVP).

**Преимущества:** Один раз написал → 100+ сценариев из конфига. Интерактив. Масштабируется.

## Почему не CARLA / MetaDrive / Godot

| Вариант | Почему нет |
|---|---|
| CARLA | Требует Linux+NVIDIA GPU. Overkill для образования. Фотореализм не нужен. |
| MetaDrive | Panda3D не работает на macOS ARM. Мёртв на маках. |
| Godot без ассетов | Цветные кубики. Нужен 3D-художник. Three.js гибче для веба. |
| Чистый SUMO | Визуал — схема. Нужен фронт для красоты. |

## Рекомендуемый стек (Фаза 2)

```

│  Frontend: Next.js + R3F        │

│  ├── Точная симуляция трафика   │
│  ├── Фазы светофоров            │

│  └── Сценарии из YAML/JSON      │
└─────────────────────────────────┘
```
